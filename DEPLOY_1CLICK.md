# 🚀 DEPLOY EN 1 CLICK

## Opción 1: Vercel (Recomendado - Optimizado para Next.js)

**GitHub:** https://github.com/lucaseiros-lab/epl-analytics

### Deploy en Vercel (5 segundos)
👉 https://vercel.com/new?repo=https://github.com/lucaseiros-lab/epl-analytics

Pasos:
1. Click en el link anterior
2. Click "Deploy"
3. Esperar 1-2 minutos
4. **URL en vivo aparecerá automáticamente**

## Opción 2: Railway (Alternativa - Muy rápido)

### Deploy en Railway (3 minutos)
👉 https://railway.app?referralCode=lucaseiros

Pasos:
1. Click en "New Project" → "Deploy from GitHub repo"
2. Seleccionar: `lucaseiros-lab/epl-analytics`
3. Vercel auto-detectará Next.js
4. Click "Deploy"
5. **URL obtenida en ~2 minutos**

---

## Opción 3: Render (Otra alternativa)

### Deploy en Render (Gratis con límites)
1. Ir a: https://render.com
2. Click "+ New" → "Web Service"
3. Conectar GitHub: `lucaseiros-lab/epl-analytics`
4. Settings:
   - Name: `epl-analytics`
   - Build Command: `npm run build`
   - Start Command: `npm start`
5. Click "Deploy"
