PASO 1: Instalar dependencias
   $ npm install

PASO 2: Crear cuenta Supabase (gratis)
   → https://supabase.com
   → Copiar credenciales a .env.local

PASO 3: Ejecutar SQL
   → Copiar contenido de setup.sql
   → Pegar en Supabase SQL Editor
   → Click "Run"

PASO 4: Ejecutar app local
   $ npm run dev
   → Abre http://localhost:3000

PASO 5: Cargar datos de prueba
   → Ve a /upload
   → Sube los 6 CSVs
   → Ve a /dashboard

PASO 6: Deploy (opcional)
   → Subir a GitHub
   → Conectar con Vercel
   → Tu app en internet (gratis)
